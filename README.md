# 🎓 NAVTTC AI Course Chatbot

An AI-powered chatbot built with Streamlit that helps students explore NAVTTC courses, admission requirements, eligibility criteria, and training opportunities through a simple and interactive interface.

## 📖 Overview

The NAVTTC AI Course Chatbot is designed to provide quick access to information about vocational and technical training programs. Users can ask questions through text or voice input and receive instant responses regarding courses, admissions, duration, eligibility, and other related details.

## ✨ Features

- AI-powered chatbot
- Voice input support
- Course information retrieval
- Admission guidance
- Eligibility and duration details
- Interactive Streamlit interface
- Responsive and user-friendly design

## 🛠️ Built With

- Python
- Streamlit
- Machine Learning
- Natural Language Processing (NLP)

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/navttc-ai-chatbot.git
```

2. Navigate to the project directory:

```bash
cd navttc-ai-chatbot
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app.py
```
